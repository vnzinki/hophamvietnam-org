== Description ==

Họ Phạm Việt Nam là theme WordPress được thiết kế dành riêng cho website dòng họ,
mang phong cách văn hóa truyền thống Việt Nam với các yếu tố trang trí cổ điển:

* Rồng chạm khắc cổ (Ancient dragon carvings)
* Hoa sen, phượng hoàng (Lotus, phoenix motifs)
* Họa tiết trống đồng Đông Sơn (Dong Son bronze drum patterns)
* Nút thắt vô tận, Ngũ Phúc (Endless knot, Five Blessings)
* Tông màu vàng đồng, nâu gỗ, kem cổ (Gold, wood brown, antique cream palette)

Tính năng:
* Trang chủ tùy chỉnh với các chuyên mục nổi bật
* Thanh điều hướng cố định khi cuộn trang
* Sidebar với bài viết được đọc nhiều nhất
* Thiết kế responsive cho mọi thiết bị
* Tối ưu cho tiếng Việt

== Installation ==

1. Tải file hopham-vietnam.zip
2. Vào WordPress Admin → Giao diện → Themes → Thêm mới → Tải lên
3. Chọn file zip và nhấn "Cài đặt ngay"
4. Kích hoạt theme

Hoặc giải nén thư mục `hopham-vietnam` vào `wp-content/themes/`.
